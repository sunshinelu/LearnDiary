# Markdown使用小技巧

1. 上标：
- n的平方等于n+1，写法如下：

    `n<sup>2</sup>=n+1`

显示结果为：
n<sup>2</sup>=n+1


2. 下标：

- a=log2b，写法如下：
    `a=log<sub>2</sub>b`
    
显示结果为：
a=log<sub>2</sub>b

参考资料：
如何在markdown中打出上标、下标和一些特殊符号:
